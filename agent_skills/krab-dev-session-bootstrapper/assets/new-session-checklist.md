# New Session Checklist

- Определи текущую учётку и repo path.
- Проверь ветку и грязное дерево.
- Реши, нужен ли `dev-tools`, `runtime-admin` или `full`.
- Убедись, что sync идёт только в `~/.codex/skills` и `~/.claude/krab-agents`.
- Не трогай `~/.openclaw`, если задача только про разработку.
- Перед делегированием зафиксируй branch, ownership и запрещённые зоны записи.
