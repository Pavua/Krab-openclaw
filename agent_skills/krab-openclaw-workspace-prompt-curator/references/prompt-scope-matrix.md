# Prompt Scope Matrix

## `SOUL.md`

- Persona
- Глубокие поведенческие принципы
- Долгоживущая идентичность Краба

## `USER.md`

- Как работать с владельцем
- Приоритеты и ожидания по ответам
- Нюансы коммуникации и пользовательские привычки

## `TOOLS.md`

- Практические tool contracts
- Media/browser/file delivery rules
- Что обязательно сделать, чтобы инструментальный результат дошёл до пользователя

## Чего избегать

- Дублировать один и тот же блок в трёх файлах
- Писать в prompt то, чего нет в коде или runtime
