# Workspace Curation Checklist

## До правки

- Понятно, какой prompt-файл нужно менять
- Понятно, что подтверждено кодом или smoke
- Нет дублирующей repo-only заметки, которую проще обновить вместо runtime prompt

## После правки

- Нет противоречия с `AGENTS.md`
- Нет противоречия с live runtime truth
- Изменён минимально необходимый prompt layer
