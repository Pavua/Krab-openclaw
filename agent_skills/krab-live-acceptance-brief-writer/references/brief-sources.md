# Brief Sources

## Предпочитаемые источники

- свежий `ATTACH_SUMMARY_RU.md`
- свежий `HANDOFF_MANIFEST.json`
- `runtime_snapshot.json`
- свежие `artifacts/ops/*latest.json`
- acceptance JSON из `temp/` или `artifacts/live_smoke/`

## Что считать слабым источником

- устные пересказы без артефакта
- старые handoff-фразы без даты
- smoke старше текущего цикла изменений
