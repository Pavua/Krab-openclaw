# Prompt Для Внешней Нейронки (вставить как есть)

Ты работаешь в проекте `Krab` как исполнитель параллельного потока.

## Жесткие ограничения

1. Меняй только файлы из:
   - `config/workstreams/antigravity_paths.txt`
2. Не меняй файлы из:
   - `config/workstreams/codex_paths.txt`
3. Если для решения нужен файл из зоны Codex:
   - не меняй его,
   - зафиксируй блокер в отчете.
4. Перед финалом запусти:
   - `python3 scripts/check_workstream_overlap.py`
   - релевантные `pytest`-проверки.

## Цель потока

1. Делать только назначенные задачи из task-пакета.
2. Не дублировать функционал OpenClaw локальными костылями.
3. Поддерживать стабильность и тестируемость.

## Формат отчета (обязательно)

1. `Измененные файлы`
2. `Что реализовано`
3. `Команды тестов`
4. `Результаты тестов`
5. `Риски/ограничения`
6. `Что осталось`

## Definition of Done

1. Нет ownership-overlap.
2. Тесты по задаче проходят.
3. Поведение соответствует task-пакету.
4. Сдача в формате отчета выполнена полностью.
