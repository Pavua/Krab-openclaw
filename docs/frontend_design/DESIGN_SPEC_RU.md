# Krab Web Panel — Дизайн Спецификация & UI Система

## 1. Введение (Nano Banana Pro Design Vision)

Панель Краба должна чувствоваться как "мозг" системы — киберпанковый, но премиальный и чистый интерфейс. Мы отходим от дефолтных полупрозрачных блоков, которые "плывут" при большом количестве инпутов, к структурированному grid-дизайну с четкой иерархией.

## 2. UI-система (Design Tokens)

### 2.1. Цвета (CSS Vars)

- `--bg-base`: `#050505` — Абсолютный темный фон космоса.
- `--bg-panel`: `#0f1115` — Фон панелей (минимальный контраст).
- `--bg-surface`: `#1a1d24` — Инпуты и кнопки внутри панелей.
- `--border-subtle`: `rgba(255, 255, 255, 0.08)` — Едва заметные разделители.
- `--border-focus`: `rgba(56, 189, 248, 0.5)` — Фокус на формах.
- `--text-main`: `#f1f5f9` — Текст (не чисто белый).
- `--text-muted`: `#94a3b8` — Вторичные элементы, подписи.
- `--accent-cyan`: `#0ea5e9` — Основной брендовый цвет, glow-эффекты.
- `--accent-purple`: `#8b5cf6` — Цвет аналитики/deep research.
- `--state-ok`: `#10b981` — Зеленый (успех/ON).
- `--state-warn`: `#f59e0b` — Желтый (alert).
- `--state-bad`: `#ef4444` — Красный (ошибка/OFF).

### 2.2. Типографика

- **Font-Family**: `'Outfit', system-ui, sans-serif`
- **Заголовки панелей**: `16px`, `600` (semibold), uppercase `letter-spacing: 0.05em`.
- **Основной текст**: `14px`, `400`.
- **Значения метрик**: `28px`, `700`, моноширинные цифры.

### 2.3. Структура сетки (Grid)

Используем 12-колоночную сетку CSS Grid.

- **Top Row (Metrics)**: Вывод 4-6 ключевых показателей блоками в один ряд (адаптивно перетекают).
- **Middle Row (Context)**: Слева — сервисы/линки (4 cols), справа — Ops/Alerts (8 cols).
- **Core Action (Assistant)**: Занимает 100% ширины, разбит внутри на:
  1. Config-панель сверху.
  2. Input-сценариев.
  3. Action bar + quick utils.

## 3. Компоненты (Nano UI Components)

- **.glass-panel**: Бордер в 1px с легким градиентом границы (`linear-gradient(bottom right)`), внутренний padding 20px, `border-radius: 12px`.
- **.btn-primary**: Не просто заливка, а `background: linear-gradient(135deg, #0284c7, #0ea5e9)`, `box-shadow: 0 4px 14px 0 rgba(14,165,233, 0.39)`.
- **.btn-ghost**: `background: transparent`, border 1px subtle, :hover меняет на solid bg-surface.
- **.input-field**: Высота 40px, скругления 8px. Эффект свечения на `:focus`.
- **.badge**: Маленький pill (скругление 12px). Свечение текста под цвет бэйджа.

## 4. Специфичные UX Улучшения

- **Состояния пустых данных / загрузки**: Замена "—" на `.skeleton-text` с анимацией пульса.
- **Интерактивные селекты**: `select` переделаны по стилям, чтобы не выбиваться.
- **Контроль ошибок**: Сообщения выводятся внутри панелей в `.alert-box`.
- **Output-зона ассистента**: Задать скролл с кастомным `::-webkit-scrollbar`, фон чуть темнее (`#090A0C`), шрифт - моно (для кода) + поддержка markdown-окружения через стили `pre`/`code`.

## 5. Прототипирование

Анимации стейтов (loading -> render data) описаны CSS transitions. Прототип `index_redesign.html` показывает структуру сетки и UI-кит.
