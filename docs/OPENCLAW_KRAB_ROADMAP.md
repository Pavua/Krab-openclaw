# OpenClaw Krab Roadmap

## Canonical execution order

1. `Truth Reset`
2. `OpenClaw Stability Kernel`
3. `Channel Reliability / Proactive Core`
4. `System / Browser / Capability Expansion`
5. `Multimodal + Voice Foundation`
6. `Ordinary Call Translator MVP`
7. `Translator Daily-Use Hardening`
8. `Monetization Layer`
9. `Product Teams / Swarm / Controlled Autonomy`

## Что считать done только после acceptance

- routing/auth/restart/panel truth;
- multi-account freeze/reclaim;
- translator live preflight;
- ordinary-call daily-use;
- release-safe runtime.

## Что не делать раньше времени

- не раздувать product-team и swarm-треки раньше stable core;
- не считать monetization первой delivery-целью;
- не смешивать personal-primary и release-safe truth.
